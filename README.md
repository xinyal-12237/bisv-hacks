# bisv-hacks

This is our text-to-speech and speech-to-text translate app for communication! By using this app, the user can remove the language barrier and gain access to opportunities. With accurate, high-quality, and humanlike speech and translation, one can clearly convey their meaning in a language they don't know. This can help people who might not be able to access opportunities get a chance to prove their qualifications and overcome obstacles that would have jepoardized their career.
